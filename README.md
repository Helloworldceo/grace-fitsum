# Grace Fitsum Portfolio

A high-end, single-page bilingual (English + Amharic) wedding and event portfolio, designed for GitHub Pages.

## Local Preview

Open `index.html` directly in your browser, or run a local server:

```bash
cd grace-fitsum-portfolio
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Launch on GitHub Pages (Fastest)

1. Create a new GitHub repository (for example: `grace-fitsum-portfolio`).
2. Push this folder to the repository root.
3. In GitHub, open **Settings > Pages**.
4. Under **Build and deployment**, choose:
   - **Source**: Deploy from a branch
   - **Branch**: `main` and `/ (root)`
5. Save. Your site will be live in about 1-3 minutes.

Expected URL format:

`https://<your-github-username>.github.io/grace-fitsum-portfolio/`

## Custom Domain (Optional)

1. In **Settings > Pages**, add your custom domain.
2. In your DNS provider, create the required records shown by GitHub.
3. Enable **Enforce HTTPS** once certificate provisioning is complete.

## Content Customization

- Replace hero portrait placeholder by editing the `background` URL in `style.css`.
- Update phone number in the contact section in `index.html`.
- Replace gallery placeholders with real project images and captions.
